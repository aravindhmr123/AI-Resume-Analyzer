🧠 AI Resume Analyzer

An AI-powered web application that analyzes resumes using LLMs (via Groq API) to provide resume scoring, keyword matching, and intelligent feedback


🚀 Features

📄 Upload resume (PDF / DOC / DOCX)

🤖 AI-powered analysis using Groq LLM

🎯 Resume vs Job Description matching

📊 Resume scoring (ATS-style + AI insights)

🔍 Extract:

Key skills

Missing keywords

💡 Smart AI suggestions for improvement

⚡ Fast response using Groq inference



 🛠️ Tech Stack

* Frontend: React.js
* Backend: Flask (Python)
* AI: Groq API (LLM)
* Parsing: pdfplumber, python-docx

---

 ⚙️ Setup

# Backend

```
cd backend
pip install -r requirements.txt
python -m spacy download en_core_web_sm
python app.py
```

# Frontend

```
cd frontend
npm install
npm start
```
